overview: >
  StudyNotion is an all-in-one educational platform that empowers both students and instructors with tools for learning and teaching. With a responsive UI, secure backend, and rich media support, StudyNotion delivers an engaging digital education experience.

keyFeatures:
  - "Student Experience: Course discovery, wishlist, checkout, content streaming, and profile editing."
  - "Instructor Dashboard: Track insights, manage courses, view ratings, and update content."
  - "Secure Authentication: Email/password login, OTP verification, JWT, and password reset."
  - "Course Management: CRUD operations on course data and media-rich lessons."
  - "Razorpay Payment Integration: Real-time checkout flow for course purchases."
  - "Cloudinary Integration: Scalable media storage for images and videos."
  - "Markdown-Based Content: Rich course descriptions rendered via Markdown."
  - "Role-Based Access Control: Segregated permissions for students and instructors."

technologiesUsed:
  frontend:
    - "React.js – Dynamic UI for learners and instructors"
    - "Tailwind CSS – Utility-first responsive styling"
    - "Redux – Robust state management"
  backend:
    - "Node.js + Express.js – Scalable RESTful APIs"
    - "MongoDB + Mongoose – Flexible NoSQL database schema"
    - "JWT & Bcrypt – Authentication and secure password handling"
  integrations:
    - "Cloudinary – Media storage and delivery"
    - "Razorpay – Payment gateway for seamless checkout"
    - "Nodemailer – Email notifications and OTP flows"

challengesAndLearnings: >
  Building StudyNotion required implementing secure role-based routes and scalable media delivery.
  Integrating Razorpay and managing webhook callbacks posed practical challenges in transaction handling.
  Optimizing course content rendering with Markdown improved both performance and user experience.
  The project enhanced my full-stack architecture, API design, and deployment skills.

outcome: >
  StudyNotion has become a well-structured and scalable EdTech platform capable of serving real-world use cases.
  It enables instructors to share their knowledge while students access diverse courses — all through a secure,
  interactive, and user-friendly environment.

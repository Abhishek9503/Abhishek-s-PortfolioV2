## Overview

**SumoDoc** is an AI-powered document summarization platform that allows users to upload PDFs and receive beautifully structured summaries. The app leverages GPT-4 and Langchain for intelligent processing and provides a responsive and secure user experience via modern web technologies.

## Key Features

- **AI-Powered Summarization**: Converts PDFs into clear, concise summaries using GPT-4 and Langchain.
- **Interactive Viewer**: Displays summaries with progress tracking and emoji-enhanced clarity.
- **Secure File Handling**: PDF uploads up to 32MB via UploadThing with route and API protection.
- **User Dashboard**: Easily manage all uploaded summaries and track usage.
- **Markdown Export**: Turn summaries into blog-ready content.
- **Flexible Pricing**: Basic and Pro plans managed via Stripe with webhooks.
- **Real-Time Feedback**: Toast notifications for upload, processing, and error status.

## Technologies Used

- **Next.js 15 App Router**: Server Components, Server Actions, and API integration.
- **React 19**: Modern UI with reusable components.
- **Clerk Auth**: Secure sign-in with Passkeys, Google, and GitHub.
- **GPT-4 + Langchain**: For parsing, chunking, and summarizing documents.
- **NeonDB (PostgreSQL)**: Serverless data storage for summaries and users.
- **Tailwind CSS 4**: Utility-first design for responsive UI.
- **UploadThing**: Handles secure PDF uploads.
- **Stripe**: For subscription and payment management.
- **TypeScript**: Improves reliability and developer experience.
- **ShadCN UI**: Modern, accessible components.
- **Deployed on Vercel**: For fast, global delivery.

## Challenges and Learnings

Implementing real-time updates, handling large file uploads securely, and integrating GPT-4 summarization within a performant UI were key technical challenges. Working with Stripe webhooks and Clerk authentication also provided deep insights into building production-grade SaaS products.

## Outcome

SumoDoc offers a seamless solution for transforming lengthy documents into actionable summaries. It empowers users with AI-enhanced productivity while maintaining high standards for usability and security.
